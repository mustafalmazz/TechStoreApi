# TechStore API   Bu proje bir E-Ticaret Web API projesidir. **.NET Core** ve **Entity Framework Core** kullanılarak geliştirilmektedir.  ##  Teknolojiler ve Araçlar Projede aşağıdaki teknolojiler kullanılmaktadır:  * **Platform:** .NET Core (Web API) * **ORM:** Entity Framework Core (Code First Yaklaşımı) * **Veritabanı:** SQL Server   ##  Kurulum ve Çalıştırma  Bu projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:  1.  **Projeyi Klonlayın:**     ```bash     git clone [https://github.com/mustafalmazz/TechStoreApi.git](https://github.com/KULLANICI_ADIN/TechStoreApi.git)     cd TechStoreApi     ```  2.  **Veritabanı Ayarları:**     `appsettings.json` dosyasındaki Connection String alanını kendi yerel veritabanı sunucunuza göre düzenleyin.  3.  **Veritabanını Oluşturun (Migration):**     Terminali proje dizininde açın ve şu komutu çalıştırın:     ```bash     dotnet ef database update     ```  4.  **Projeyi Başlatın:**     ```bash     dotnet run     ```  ##  Yol Haritası   -  Proje iskeletinin oluşturulması - Product ve Category modellerinin tasarlanması - Controller ve CRUD işlemlerinin yazılması - DTO (Data Transfer Object) dönüşümleri - Hata yönetimi (Global Error Handling) -  Swagger dokümantasyonu  *Bu proje geliştirme aşamasındadır.*
